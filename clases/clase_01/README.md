# Clase 1

Esta sesión introduce el proyecto "proyecto_sia" trabajado en clase. Se explica la estructura del proyecto, los datos utilizados y los componentes del backend que permiten cargar y consultar la información de estudiantes, cursos y matrículas.

## Contenido principal

- Datos de ejemplo en formato CSV (cursos, estudiantes, matrículas).
- Código fuente del backend para gestión de la base de datos y lógica de la aplicación.
- Pruebas unitarias básicas para el backend.
